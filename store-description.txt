Precog extracts structured context from the page you're on and assembles a Claude prompt from modular instruction blocks.

Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux) on any webpage to open a command palette. Precog reads the current page and presents a set of instruction blocks: summarize, identify TODOs, draft a reply, create an Asana task, search for related context across Gmail and Slack, or do deep research. Blocks can be combined. Precog assembles the page context and selected instructions into a prompt and sends it to Claude.

Example: open Precog on a Gmail thread and select the following blocks:

- Search for deep context: Claude searches Gmail, Google Drive, and Slack for related threads, documents, and messages, so the response is informed by the full history rather than just the current email.
- Identify TODOs: extracts TODOs from the thread. In "advise" mode, it also suggests approaches with confidence levels.
- Draft a reply: generates a response based on the thread content and any context found in the search.
- Create Asana task: creates a task to track follow-up, with a title, description, and link back to the email.

Precog has specific integrations for Gmail, Asana, and Slack. On Gmail it reads the full thread with subjects, senders, and dates. On Asana it pulls the task title and description with formatting and links preserved. On Slack it gets the message text, sender, channel, timestamp, and permalink. On any other page, Precog captures the page title, URL, and any selected text or page content.

For best results, enable connectors in Claude for Gmail, Google Drive, Slack, Asana, and any other services relevant to the context being processed.

Precog runs entirely in the browser. There is no server, no analytics, and no tracking. Data is only sent to Claude when a prompt is explicitly submitted.

Open source (Apache 2.0): https://precog.cc
